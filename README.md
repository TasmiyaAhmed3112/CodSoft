# CodSoft Python Programming Internship

This repository contains the projects I developed during my **Python Programming Internship at CodSoft**.

The internship helped me improve my Python programming skills by building practical GUI-based applications and implementing core programming concepts.

## Projects

### Task 1: Password Generator 🔐

A Python-based password generator application that creates strong and secure passwords based on user preferences.

**Features:**
- Generate random passwords
- Customize password length
- Include uppercase letters, lowercase letters, numbers, and special characters
- Simple and user-friendly interface

**Technologies Used:**
- Python
- Tkinter
- Random module

---

### Task 2: To-Do List Application ✅

A GUI-based task management application to help users organize and track their daily tasks.

**Features:**
- Add new tasks
- Delete completed tasks
- Manage daily activities
- Interactive graphical interface

**Technologies Used:**
- Python
- Tkinter

---

### Task 3: Contact Book 📱

A contact management application that allows users to store and manage contact details.

**Features:**
- Add new contacts
- Update existing contacts
- Delete contacts
- Search contacts by name
- Display contacts using a table interface

**Technologies Used:**
- Python
- Tkinter
- Treeview
